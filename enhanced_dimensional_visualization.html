🧮 Mathematical Insights & Theory

🎯 Encoding Dimension Impact

Higher dimensions → Better gradient flow:

• Binary-5: Limited expressiveness, discrete gradients

• One-Hot-26: Sparse but interpretable

• Embedding-128: Rich representations, smooth gradients

• Embedding-512: Maximum capacity, potential overfitting

⚡ Gradient Analysis

Gradient norm scaling:

∇L ∝ √d where d = encoding dimension

Convergence rate:

Binary-5: O(n²) • One-Hot: O(n√n)

Embed-128: O(n log n) • Embed-512: O(n)

🔮 Permutation Learning

Sinkhorn iterations scale with dimension:

Higher dimensions need more normalization steps

Hungarian assignment:

O(n³) complexity independent of encoding

Straight-through estimator critical for gradients

📊 Information Theory

Channel capacity increases:

C = log₂(d) bits per symbol

• Binary-5: 2.32 bits • One-Hot: 4.70 bits

• Embed-128: 7.0 bits • Embed-512: 9.0 bits

But diminishing returns after 128D
