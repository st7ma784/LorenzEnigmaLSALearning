Mathematical Theory & Equations

1. Lorenz Mask Generation

Binary Encoding: Each character C → 5-bit representation

C ∈ {A,B,...,Z} → B₅(C) = [b₄, b₃, b₂, b₁, b₀] where C = Σᵢ bᵢ·2ⁱ

Lorenz Mask Extraction:

M = P ⊕ C = B₅(plaintext) ⊕ B₅(ciphertext)

Where ⊕ is bitwise XOR operation, creating the Lorenz-style cipher mask

Mask Statistics:

Entropy(M) = -Σᵢ p(mᵢ) log₂ p(mᵢ)
Balance = |0.5 - (Σ mᵢ)/|M||

2. Permutation Matrix Representation

Hard Permutation Matrix:

P ∈ {0,1}²⁶ˣ²⁶, Σⱼ Pᵢⱼ = 1, Σᵢ Pᵢⱼ = 1 (doubly-stochastic)

Rotor Position with Offset Identity:

R(pos) = [rᵢⱼ] where rᵢⱼ = δᵢ,⟨j+pos⟩₂₆
δ = Kronecker delta, ⟨·⟩₂₆ = modulo 26

Effective Rotor Transformation:

T_effective = R(pos) × P_base × R(-pos)

Combines base wiring with position offset

3. Differentiable Approximation

Soft Permutation via Sinkhorn:

S₀ = softmax(L/τ), where L = learnable logits, τ = temperature
Sₖ₊₁ = normalize_rows(normalize_cols(Sₖ))

Sinkhorn Iteration:

S^(k+1) = diag(u) S^(k) diag(v)
where uᵢ = 1/Σⱼ S^(k)ᵢⱼ, vⱼ = 1/Σᵢ S^(k)ᵢⱼ

Hard Assignment via Hungarian:

P_hard = argmin_P ⟨-S_soft, P⟩ = Hungarian(-S_soft)

4. Gradient Computation

Loss Function:

L_total = L_reconstruction + λ₁L_doubly_stochastic + λ₂L_entropy + λ₃L_orthogonal

Reconstruction Loss:

L_reconstruction = ||M_predicted - M_target||²_F

Doubly-Stochastic Regularization:

L_doubly = Σᵢ(Σⱼ Sᵢⱼ - 1)² + Σⱼ(Σᵢ Sᵢⱼ - 1)²

Gradient Flow (Straight-Through Estimator):

∇_L L = ∇_S_soft L + α(P_hard - S_soft)
where α controls hard/soft gradient mixing

Position Gradient:

∇_pos L = Tr(∇_T L^T × ∂R(pos)/∂pos × P_base)

5. Multi-Sample Training Mathematics

Batch Gradient Averaging:

∇_batch = (1/N) Σᵢ wᵢ ∇_sample,i
where wᵢ = f(entropy_i, length_i) is sample weighting

Position Correlation Analysis:

ρ(pos, mask) = Cov(pos, mask) / (σ_pos × σ_mask)
Cov(X,Y) = E[(X-μₓ)(Y-μᵧ)]

Stepping Sequence Modeling:

pos_t+1 = (pos_t + step_pattern(t)) mod 26
T_t = R(pos_t) × P_base × R(-pos_t)

6. Optimization & Convergence

Temperature Annealing:

τ(epoch) = max(τ₀ × decay^epoch, τ_min)

Gradient Clipping:

∇_clipped = ∇ × min(1, clip_norm / ||∇||₂)

EMA Parameter Update:

θ_ema,t = β × θ_ema,t-1 + (1-β) × θ_t

Learning Rate Schedule:

lr(epoch) = lr₀ × cos(π × epoch / max_epochs)

7. Key Mathematical Insights

🔑 Permutation ↔ Doubly-Stochastic: Every permutation matrix is doubly-stochastic, enabling differentiable relaxation
⚙️ Offset Identity Matrices: Enable learnable rotor positions while preserving permutation structure
🎯 Sinkhorn Convergence: Guarantees doubly-stochastic properties with O(n²) iterations
📊 Multi-Sample Averaging: Reduces gradient variance by O(1/√N) where N = samples per config
🔄 Straight-Through Gradient: Allows backprop through discrete assignments via continuous relaxation
